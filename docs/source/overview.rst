Overview
========

이 문서는 ``autumn63/practice`` (또는 ``teampl``) 저장소의 전체 구조와 목적을 설명합니다.

- main 브랜치: 문서(Sphinx)와 예제 코드(lumache.py)가 포함되어 있습니다.
- text 브랜치: 텍스트 처리 관련 코드가 포함될 예정입니다.
- imag 브랜치: 이미지 처리 관련 코드가 포함될 예정입니다.
- audio 브랜치: 오디오 처리 관련 코드가 포함될 예정입니다.
- video 브랜치: 비디오 처리 관련 코드가 포함될 예정입니다.

각 브랜치의 자세한 내용은 :doc:`branches` 문서를 참고하세요.
